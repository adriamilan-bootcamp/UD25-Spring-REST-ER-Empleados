# UD25-Spring-REST-ER-Empleados

## URIs endpoints: 
```
Departamentos
GET       /api/departamentos
POST      /api/departamento
GET       /api/departamento/{id}
PUT       /api/departamento/{id}
DELETE    /api/departamento/{id}

Empleados
GET       /api/empleados
POST      /api/empleado
GET       /api/empleado/{id}
PUT       /api/empleado/{id}
DELETE    /api/empleado/{id}
```
